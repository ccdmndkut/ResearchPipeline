Feature Breakdown

Build on Northflank

Northflank enables continuous integration of all your code, from all your favourite version control platforms.

Implement Continuous Integration

Native VCS Compatibility

Integrate with GitHub, GitLab, or Bitbucket. Northflank syncs with your repositories, branches, and pull requests.

Self-Hosted & Enterprise VCS

Seamlessly integrate with self-hosted GitLab or Enterprise GitHub for advanced use cases.

Build on Commit

Push to your repository to trigger a build. View status via your VCS provider and pause CI at any time.

Customise Your Build Process

Dockerfile or Buildpack

Simply add a Dockerfile or Buildpack. Custom or Heroku buildpacks can be used via Git or the Heroku repository.

Monorepo Support

Contain all your services in one place with multiple Dockerfiles and build contexts in a single repository.

Multistage Builds

Build specific Dockerfile stages for faster, smaller builds. Leverage external base images from private registries.
